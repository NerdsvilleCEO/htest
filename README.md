# htest
golang http unit test

```text
Usage of ghtest:
  -c string
    	config file or path (default "./config")
  -check
    	check config file only
  -curl
    	output curl command
  -example
    	generate example config
  -ip string
    	testing ip
  -method
    	check support compare method
  -new string
    	create new config (default "http://a.com/b")
  -raw
    	raw response output
  -t string
    	template file or path (default "./template")
  -timeout string
    	overwrite timeout value
  -vv
    	verbose output
```
